
### User Stories

**As a Participant, I want to:**
*   ...be able to create an account and build my profile so that I can showcase my skills.
*   ...be able to see a list of all available hackathons so I can decide which one to join.
*   ...be able to create or join a team so I can collaborate with others.
*   ...be able to submit my project so that it can be judged.
*   ...be able to request help from a mentor when I'm stuck.

**As a Judge, I want to:**
*   ...be able to see all the projects that have been submitted for a hackathon.
*   ...be able to score each project based on a clear set of criteria.
*   ...be able to provide constructive feedback to the teams.

**As a Mentor, I want to:**
*   ...be able to see a list of teams that have requested help.
*   ...be able to provide guidance and support to the teams.

**As an Organizer, I want to:**
*   ...be able to create and manage a new hackathon.
*   ...be able to invite judges and mentors to the event.
*   ...be able to track the progress of the hackathon and see the results.

**As an Admin, I want to:**
*   ...be able to manage all users and their roles.
*   ...be able to ensure the smooth operation of the platform.
